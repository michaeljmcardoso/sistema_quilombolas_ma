# Sistema de Acompanhamento: Regularização Quilombola MA

Dashboard para visualização e controle do andamento dos processos de regularização das comunidades quilombolas no Maranhão.

## 🛠️ Tecnologias Utilizadas

- **Python**: Linguagem de programação
- **Streamlit**: Interface web interativa
- **SQLite**: Banco de dados leve e local
- **Plotly**: Visualizações e gráficos interativos
- **Pandas**: Manipulação de dados

## 📋 Pré-requisitos

Python 3.8 ou superior instalado.
